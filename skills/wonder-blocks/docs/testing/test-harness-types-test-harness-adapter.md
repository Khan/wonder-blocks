# TestHarnessAdapter\<>

```ts
type TestHarnessAdapter<TConfig> = (
    children: React.Node,
    config: TConfig,
) => React.Element<any>;
```

This type represents the signature of a an adapter function for use with the test harness framework methods, [`makeTestHarness()`](./?path=/docs/packages-testing-test-harness-exports-maketestharness--docs) and [`makeHookHarness()`](./?path=/docs/packages-testing-test-harness-exports-makehookharness--docs).
