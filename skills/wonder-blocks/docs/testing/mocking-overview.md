# Mocking

Wonder Blocks Testing provides support for mocking the Wonder Blocks Data GraphQL fetching via the [`mockGqlFetch()`](./?path=/docs/packages-testing-mocking-exports-mockgqlfetch--docs) export, and `fetch()`-like methods via the [`mockFetch()`](./?path=/docs/packages-testing-mocking-exports-mockFetch--docs) export. These are both supported by the [`RespondWith`](./?path=/docs/packages-testing-mocking-exports-respondwith--docs) API for easily generating the responses.
