## Usage

The typography tokens are used to define the font family, size, line height and
weight of text elements in our components.

```tsx
import {font} from "@khanacademy/wonder-blocks-tokens";
const styles = {
    fontSize: font.size.large,
    lineHeight: font.lineHeight.large,
    fontWeight: font.weight.regular,
    fontFamily: font.family.serif
};
```
