# FixtureProps

```ts
type FixtureProps<TProps: {...}> =
    | $ReadOnly<TProps>
    | ((options: $ReadOnly<GetPropsOptions>) => $ReadOnly<TProps>);
```

This type describes how props can be provided to a fixture as either a plain object, or a function that returns a plain object. The function will be called with an API to assist in generating the props (see <a href="./?path=/docs/packages-testing-fixtures-types-getpropsoptions--docs">`GetPropsOptions`</a>).
