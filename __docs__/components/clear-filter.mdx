# Clear filter

## Clear filters

<NeutralBadge label="Bulk state reset" /> Ensures users can return to a neutral dataset instantly without manual deselection, regardless of how many components are currently in use.

<img src={clearFiltersImg} alt="Clear filters" />

## When to use

The choice to use a clear filters button is determined by the need to provide a fast escape hatch for complex queries or empty result states.

Use when

A single selection narrows the data

A filter combination returns an empty state

Dismissing multiple active filters individually would be time-intensive

Example use

Selecting a single school from a dropdown menu.

Clearing all criteria after "0 results" are found for specific teacher/grade combinations.

Returning to a full class report after narrowing by three specific grades.

## When NOT to use

Don’t use when

Clearing a required filter with no empty state

Resetting a data entry form

Recommended alternative

Dropdown selection

Individual field editing

Rationale

Do not "clear" required parameters (like Date) that cannot have a "none" state. The value should instead revert to its system default.

Do not implement "Clear all" functionality on forms. While "Clear filters" is a helpful escape hatch for viewing data, a "Reset form" button leads to accidental data loss. To ensure a safe user experience, avoid any global action that wipes unsaved user input.

## States

### Hidden (inactive)

The button should not be visible if no filters are applied, preventing UI clutter.

<img src={states1Img} alt="States 1" />

### Default (active)

Appears as a small tertiary button once a filter is triggered.
Avoid using red or other high-caution colors. Clearing filters is an inconsequential action that does not delete data; use neutral styling to keep focus on primary actions.

<img src={states2Img} alt="States 2" />

## Placement

Position the "Clear filters" button directly following the list of active filter chips.
It should sit between the filter bar (controls) and the data (results), providing a clear bridge between the two.
