# Khanmigo

## Khanmigo

Khanmigo is Khan Academy’s AI-powered educational mascot that serves as a multi-faceted companion. Its primary function is to act as a guide for students and an assistant for teachers and administrators.

<img src={khanmigoImg} alt="Khanmigo" />

## When to use

The choice to use Khanmigo is determined by the need to provide interactive pedagogical guidance, administrative assistance, or motivational engagement during non-evaluative learning activities.

### Active tutoring, providing hints or guidance during practice/assignments

Guiding students through their learning journey and encouraging rewards (motivational wayfinding)

Giving educators (teacher/admins) student insights or classroom management shortcuts

Helping a student solve a linear equation in the "Practice" frame.

Motivating students to earn gems and unlock new mascot accessories.

Teachers clicking the persistent "Ask Khanmigo Assistant" button in the global header to open a chat interface or any ask Khanmigo input field.

## When NOT to use

### Replacing core navigation elements

### Communicating critical system alerts

### A student is in a summative assessment

### Offering personalization in the teacher role

Tabs or pills

Standard modals or banners

Khanductor (or no mascot)

Fixed mascot state

Do not use the mascot as a replacement for primary navigation elements like the "Reports" or "Assignments" tabs.

Avoid using the mascot for critical system errors or technical outages; these require a serious, neutral tone.

Do not use Khanmigo when a student is taking a quiz, unit test, or any high-stakes assessment. Use Khanductor for the assessment product. Khanmigo’s tutoring role conflicts with the need for independent student performance.

In the Teacher/Admin role, do not offer accessory customization; it must remain a professional "Assistant" persona.

## Roles and versioning

Learning Coach: Interactive 1:1 tutoring and personalized support.  Motivation guide: navigation and gem motivation.

### Motivational Guide: Navigation and gem motivation only.

### Concierge/Assistant: Direct Q&A, instructional tools, and class insights.

### Concierge (Limited): Basic assistance tools.

AI-powered: Full generative support.

Static: No generative AI tutoring capabilities.

## Placement

### Practice/chat frame (learners)

Embedded directly into the student learning view or as a dedicated side-panel for natural language discussion.

### Global header (teacher/admin)

A persistent "Ask Khanmigo Assistant" button is located in the top-right of the interface for teachers.

### Concierge frame (teacher/admin)

Appears in the teacher dashboard to provide "Today's insights" regarding student struggle or activity.

## Scale

### Student scale

Khanmigo is always presented at a large scale to emphasize its role as a prominent, friendly companion. With the exception when chatting on mobile or zoomed in.

### Teacher/admin scale

Khanmigo has a small version that can be used integrated within functional UI elements like buttons or search fields. This small version is a unique component found here. DO NOT use big Khanmigo and scale down.

## Visualization and customization

### Color selection

Learners can customize the color of Khanmigo between 4 colors options. Semantic tokens: Learning > Characters > Flesh Blue / Cyan / Green / Magenta / Orange

### Accessories

### Learners can personalize Khanmigo with various hats and glasses.

### Special migos

Learners can unlock special, unique versions of Khanmigo. These too can be customized through color and accessories. Khanmigo is always retained in the name, so the example here is Cat Khanmigo.

### No customization for teachers or admins

Khanmigo remains in a fixed, professional "Assistant" state for the teachers and admins experience.

### No alternative characters

To ensure a consistent user experience, Khanmigo serves as the exclusive AI persona for all users. It is the sole guide and tutor for students across the experience—including the dashboard, learning queue, and all practice modes—with the exception of assessments. Khanmigo is also the permanent assistant for teachers and administrators. Using any character other than Khanmigo is strictly prohibited.

## Interaction

### Dynamic animation

Khanmigo animates across various states—including idle breathing, thinking while processing AI responses, and celebratory motions—to provide immediate feedback and presence.

### Visual signaling overrides

There are instances where Khanmigo must visually portray a specific signal (such as a milestone achievement or a time-sensitive reminder). In these moments, a special version of the mascot may temporarily override a student's personal customization to ensure the signal is clearly communicated.
