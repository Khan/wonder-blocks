# Khanmigo

## Header

### Khanmigo

AI learning coach/AI assistant

Khanmigo is Khan Academy’s AI-powered educational mascot that serves as a multi-faceted companion. Its primary function is to act as a guide for students and an assistant for teachers and administrators.

<img src={headerImg} alt="Header" />

## Table

### When to use

The choice to use Khanmigo is determined by the need to provide interactive pedagogical guidance, administrative assistance, or motivational engagement during non-evaluative learning activities.

| Use when | Example use |
| --- | --- |
| Active tutoring, providing hints or guidance during practice/assignments | Helping a student solve a linear equation in the "Practice" frame. |
| Guiding students through their learning journey and encouraging rewards (motivational wayfinding) | Motivating students to earn gems and unlock new mascot accessories. |
| Giving educators (teacher/admins) student insights or classroom management shortcuts | Teachers clicking the persistent "Ask Khanmigo Assistant" button in the global header to open a chat interface or any ask Khanmigo input field. |

## Table

### When NOT to use

| Don’t use when | Recommended alternative | Rationale |
| --- | --- | --- |
| Replacing core navigation elements | Tabs or pills | Do not use the mascot as a replacement for primary navigation elements like the "Reports" or "Assignments" tabs. |
| Communicating critical system alerts | Standard modals or banners | Avoid using the mascot for critical system errors or technical outages; these require a serious, neutral tone. |
| A student is in a summative assessment | Khanductor (or no mascot) | Do not use Khanmigo when a student is taking a quiz, unit test, or any high-stakes assessment. Use Khanductor for the assessment product.  Khanmigo’s tutoring role conflicts with the need for independent student performance. |
| Offering personalization in the teacher role | Fixed mascot state | In the Teacher/Admin role, do not offer accessory customization; it must remain a professional "Assistant" persona. |

## Table

### Roles and versioning

| Version | Role for students | Role for teachers/admins | AI Capabilities |
| --- | --- | --- | --- |
| Full (licensed) | Learning Coach: Interactive 1:1 tutoring and personalized support.  Motivation guide: navigation and gem motivation. | Concierge/Assistant: Direct Q&A, instructional tools, and class insights. | AI-powered: Full generative support. |
| Freemium | Motivational Guide: Navigation and gem motivation only. | Concierge (Limited): Basic assistance tools. | Static: No generative AI tutoring capabilities. |

## Visualization and customization

### Color selection

Learners can customize the color of Khanmigo between 4 colors options.
Semantic tokens: Learning > Characters > Flesh Blue / Cyan / Green / Magenta / Orange

<img src={visualizationAndCustomization1Img} alt="Visualization and customization 1" />

### Accessories

Learners can personalize Khanmigo with various hats and glasses.

<img src={visualizationAndCustomization2Img} alt="Visualization and customization 2" />

### Special migos

Learners can unlock special, unique versions of Khanmigo. These too can be customized through color and accessories.
Khanmigo is always retained in the name, so the example here is Cat Khanmigo.

<img src={visualizationAndCustomization3Img} alt="Visualization and customization 3" />

### No customization for teachers or admins

Khanmigo remains in a fixed, professional "Assistant" state for the teachers and admins experience.

<img src={visualizationAndCustomization4Img} alt="Visualization and customization 4" />

### No alternative characters

To ensure a consistent user experience, Khanmigo serves as the exclusive AI persona for all users. It is the sole guide and tutor for students across the experience—including the dashboard, learning queue, and all practice modes—with the exception of assessments. Khanmigo is also the permanent assistant for teachers and administrators. Using any character other than Khanmigo is strictly prohibited.

<img src={visualizationAndCustomization5Img} alt="Visualization and customization 5" />

## Interaction

### Dynamic animation

Khanmigo animates across various states—including idle breathing, thinking while processing AI responses, and celebratory motions—to provide immediate feedback and presence.

<img src={interaction1Img} alt="Interaction 1" />

### Visual signaling overrides

There are instances where Khanmigo must visually portray a specific signal (such as a milestone achievement or a time-sensitive reminder). In these moments, a special version of the mascot may temporarily override a student's personal customization to ensure the signal is clearly communicated.
