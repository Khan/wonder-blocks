# Spacing

<Subtitle>
    <ComponentInfo name={name} version={version} />
</Subtitle>

Spacing is a collection of names assigned to specific dimensions. These are used
frequently when laying out Wonder Blocks components (such as with
[View](/docs/core-view--default)). You can use these sizes directly by
importing the `wonder-blocks-spacing` package and accessing the named property
like so: `Spacing.xxSmall_6`.

<SpacingTable />

## Usage

```js
import Spacing from "@khanacademy/wonder-blocks-spacing";

const styles = StyleSheet.create({
    box: {
        padding: Spacing.xxSmall_6,
    },
});
```
