# ErrorOptions

```ts
type ErrorOptions = {|
    metadata?: ?Metadata,
    cause?: ?Error,
|};
```

These options allow for the provision of a causal error instance as well as additional metadata that may be useful to the specific error being constructed (such as <a href="./?path=/docs/packages-data-exports-dataerror--docs">`DataError`</a> or <a href="./?path=/docs/packages-data-exports-gqlerror--docs">`GqlError`</a>).
