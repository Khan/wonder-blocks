# SharedCache

The `SharedCache` export can be used to view and modify the in-memory cache used by <a href="./?path=/docs/packages-data-exports-usesharedcache--docs">`useSharedCache`</a> hook and the hooks and components that relate to it.

The `SharedCache` export implements the <a href="./?path=/docs/packages-data-types-scopedcache--docs">`ScopedCache` interface type</a>.
