Button example:
```js
<Button
    onClick={(e) => console.log("Hello, world!")}
>Label</Button>
```

Button example, `href="#button-1"`:
```js
<Button
    onClick={(e) => console.log("Hello, world!")}
    href="#button-1"
>Label</Button>
```

Disabled button example:
```js
<Button
    onClick={(e) => console.log("Hello, world!")}
    disabled={true}
>Label</Button>
```
